Projetos

JOGO DA MEMÓRIA

Coloca à prova a memória do jogdor: este jogo embaralha pares de palavras e as esconde em cartões, as quais o jogador deve selecionar corretamente um par por vez até que todos os pares estejam visíveis. O jogo permite várias tentativas por jogador e mostra o ranking das melhores pontuações.

Ferramentas Utilizadas: HTML , CSS e JAVASCRIPT

PROCESSADOR MONOCICLO

Busca e executa instruções: capaz de somar, subtrair, dividir e multiplicar, além de ser capaz de salvar, ler e carregar os dados do banco de registradores, realizando uma instrução por clock.

Ferramentas Utilizadas: JAVA

SIMULADOR DE MEMÓRIA CACHE

Simula a busca em memória cache: ao realizar uma operação de leitura ou escrita, o programa verifica se o endereço referenciado está entre os mais recentemente acessados (memória cache). Caso esteja, o tempo de resposta é para esta instrução é x, mas caso não o encontre, ele busca na memória principal, substituindo o endereço menos acessado na cache pelo endereço trazido da memória principal, fazendo esta instrução ter o tempo de resposta x + y.

Ferramentas Utilizadas: C++
